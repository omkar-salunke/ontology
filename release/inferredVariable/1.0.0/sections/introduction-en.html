Introduction back to ToC

The inferred variable ontology defines the different classes of variable types that can be inferred from measured variables in a paleo climate dataset. This ontology is the product of an evolving community effort, crowdsourced through the Linked Earth wiki.

The inferred variable ontology extends the Linked Earth Core Ontology.
